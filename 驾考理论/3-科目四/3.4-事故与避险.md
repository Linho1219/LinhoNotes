# 事故与避险

## 关键词

- **事故避险包括：==爆胎、侧滑、碰撞、制动失效、转向失控==**
- **正确操作：==稳方向、抢挂低挡、不急转、不急刹==**
- 只有事故无法避免，可紧急制动

## 常见考点

### 转向失控

- 转向困难，选择安全地点停车
- 果断连续踩松制动踏板
- **不可紧急制动**

### 制动失效

- 先减挡，发动机制动、避险车道、刮擦
- **禁止越二级减挡**（5→2 可以，5→3 不行）
- **禁止首先拉紧驻车制动**（手刹）

### 碰撞

- 向碰撞一侧修正方向（反打容易侧翻）
- **车速较高/高速，可==先==制动**（稳住方向→减速→转方向）
- 正面碰撞侧面躲，**不跳车**（很容易受伤）

### 侧滑

- 泥泞影响选侧滑
- 修正：前轮相反，后轮相同
- 停车观察、驱动轮下铺石

::: example

【多选】==转向失控==后，如果机动车偏离直线行驶方向，应怎样使机动车尽快减速停车？

**A.** 紧急制动<br>**B.** 抢挂低速挡<br>**C.** 合理使用行车制动和驻车制动，避免紧急制动<br>**D.** 开启危险报警闪光灯

---

【答案】**BCD**

:::

::: example

【多选】下坡路行驶，==制动突然失效==后，可采取的减速方法是什么？

**A.** 利用道路边专设避险车道减速停车<br>**B.** 用车身靠向路旁的岩石或树木碰擦<br>**C.** 首先拉紧驻车制动<br>**D.** 抢挂低速挡

---

【答案】**ABD**

:::

::: example

在==车速较高==可能与前方机动车发生==碰撞==时，驾驶人应采取什么措施？

**A.** 先制动减速，后转向避让<br>**B.** 急转方向向左避让<br>**C.** 急打方向，向右避让<br>**D.** 先转向避让，后制动减速

---

先稳方向，再减速，后转向避让。速度太快时打方向容易翻车。

【答案】**A**

:::

::: example

驾驶机动车在高速公路意外==碰撞==护栏时，应迅速==向相反的方向==转向修正。

**A.** 正确<br>**B.** 错误

---

应向碰撞一侧修正。

【答案】**B**

:::

::: example

在泥泞路段遇车==后轮向右侧滑==时应如何处置？

**A.** 继续加速<br>**B.** 向右转向<br>**C.** 向左转向<br>**D.** 紧急制动

---

侧滑修正：前轮相反，后轮相同。后轮向右侧滑，则向右转向。

## 爆胎

- **胎压==过高过低==都会爆胎**
- **==先稳方向再减速==，非必要==不急刹==**

::: example

【多选】驾驶机动车==避免爆胎==的正确做法是么？

**A.** 降低轮胎气压<br>**B.** 定期检查轮胎<br>**C.** 及时清理轮胎沟槽内的异物<br>**D.** 更换有裂纹或损伤的轮胎

---

胎压过低也会爆胎，A 错误。

【答案】**BCD**

:::

::: example

轮胎胎压过高时易爆胎，因此夏季天气炎热时，应经常放气降低胎压。

**A.** 正确<br>**B.** 错误

---

胎压过低也会爆胎。

【答案】**B**

:::

::: example

【多选】机动车在高速行驶中，突然爆胎应采取的安全措施是什么？

**A.** 紧急制动，靠边停车<br>**B.** 牢牢地握住转向盘，保持直行<br>**C.** 立即松开加速踏板<br>**D.** 轻踩制动踏板

---

避免急刹，A 错误。

【答案】**BCD**

:::

::: example

前轮胎爆裂已出现转向时，驾驶人不要过度矫正，在控制住方向的情况下，应怎样做，使车辆缓慢减速？

**A.** 采取紧急制动<br>**B.** 使用驻车制动<br>**C.** 轻踏制动踏板<br>**D.** 迅速踏下制动踏板

---

【答案】**C**

:::

## 车辆故障

**==打灯 放标 人转移 报警==**

| 口诀   | 具体操作                                            |
| ------ | --------------------------------------------------- |
| 打灯   | 危险报警闪光灯                                      |
| 放标   | 车后放警告标志<br />高速 150 米以外，普路 50~100 米 |
| 人转移 | 转移到==右侧==路肩<br />==禁止留在车内等待救援==    |
| 报警   | 找救援或拖曳车                                      |

::: example

【多选】雾天驾驶机动车在==高速公路==遇事故不能继续行驶时，以下==错误==的做法是什么？

**A.** 车上人员要迅速从==左侧==车门离开<br>**B.** 在来车方向 ==100m== 处设置警告标志<br>**C.** 开启危险报警闪光灯和==远光灯==<br>**D.** 车上人员站到护栏以外安全的地方

---

应尽量从右侧车门下车，A 错误；高速公路上警告标防止在 150m 处，B 错误；非必要不开远光灯，C 错误。

【答案】**ABC**

:::

::: example

【多选】发生交通事故后，有效防止二次事故的措施是什么？

**A.** 疏散人员<br>**B.** 开启危险报警闪光灯<br>**C.** 标记伤员的原始位置<br>**D.** 正确放置危险警告标志

---

【答案】**ABD**

:::

## 着火

- **不得用水灭火**（水的密度比油大，油在水面火会越来越大）
- **不得车内等待救援**
- **==不得开启发动机罩==**（否则补充氧气）

::: example

【多选】驾驶机动车在隧道内行驶，遇前方车辆发生火灾时，以下正确的做法是什么？

**A.** 开启本车危险报警闪光灯<br>**B.** 关闭车门车窗，在车内等待救援<br>**C.** 关闭发动机，留下车钥匙，弃车逃生<br>**D.** 使用紧急电话寻求帮助

---

【答案】**ACD**

:::

::: example

**A.** 用==清水==喷洒扑灭<br>**B.** 及时报警<br>**C.** 使用车内备用的灭火器灭火<br>**D.** 在来车方向设置警告标志

---

【答案】**BCD**

:::
