# 科目四

本部分笔记基于 [驾考宝典科目四精华课](https://www.bilibili.com/video/BV17K421t7VD)。可以一同食用。

科四是拿本前的最后一道关了。又是理论，但是和科一有一定区别。科目四为 50 题，一题 2 分，90 分及格因此最多只能错 5 题。

这些**科四不考**：

- 扣分题
- 罚款题
- 判刑题
- 动画题

科一考完科四**还要考**的：交通信号。但不仅考是什么，还要考怎么做。

科四**重点要考**：案例多选题。比较难。

总的题型分分布如下

| 题型                                                                                         | 量    |
| -------------------------------------------------------------------------------------------- | ----- |
| 多选 + 案例                                                                                  | 12 题 |
| 驾驶常识<br />特殊路段、特殊天气、让行原则                                                   | 24 题 |
| 事故处置及急救<br />爆胎 故障 刹车失灵 侧滑 碰撞 着火 落水 转向失控 昏迷 失血 中毒 烧伤 骨折 | 10 题 |
| 交通信号<br />标志 标线 交警手势 信号灯                                                      | 4 题  |
| 地方题<br />个别地区考                                                                       | 2~5题 |
