# 交通信号

交通信号分四种：

- 交警手势
- 信号灯
- 交通标志
- 交通标线

这在科目四不是重点。

## 交警手势

七个交警手势仍需要掌握（[科目一 1.10](../1-科目一/1.10-信号与标志#交警手势)）：

- 停止信号：左高右低
- 直行信号：双手从未放下
- 左转弯待转信号：单左手下压
- 减速慢行信号：单右手下压
- 变道信号：单右手向中间摆动
- 转弯信号：一手掌心朝前，一手在下；哪只手在下，朝哪儿转

科目四新考法：具体情境判断。

::: example

如图所示，看到这种手势信号时怎样行驶？

![&keep-color &small](./images/cross-right-hand.png)

**A.** 在路口向左转弯<br>**B.** 停车等待<br>**C.** 在路口直行<br>**D.** 进入左弯待转区

---

此手势是右转信号，**但是交警面朝左侧，是指挥左侧车辆**。遇此情况应当**停车等候**。

::: example

【多选】驾驶机动车在隧道内行驶，以下正确的说法是什么？

**A.** 车道线为白色虚线的，不得随意变更车道<br>**B.** 车道线为白色实线的，可以变更车道<br>**C.** 车道线为白色实线的，禁止变更车道<br>**D.** 车道线为白色虚线的，禁止变更车道

---

虚线可跨，实线不可跨。C 正确。

**无论如何都不能==随意==变更车道。A 正确。**

【答案】**AC**

:::
