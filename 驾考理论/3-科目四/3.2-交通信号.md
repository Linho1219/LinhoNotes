# 交通信号

交通信号分四种：

- 交警手势
- 信号灯
- 交通标志
- 交通标线

这在科目四不是重点。

## 交警手势

七个交警手势仍需要掌握（[科目一 1.10](../1-科目一/1.10-信号与标志#交警手势)）：

- 停止信号：左高右低
- 直行信号：双手从未放下
- 左转弯待转信号：单左手下压
- 减速慢行信号：单右手下压
- 变道信号：单右手向中间摆动
- 转弯信号：一手掌心朝前，一手在下；哪只手在下，朝哪儿转

科目四新考法：具体情境判断。
