# 现场急救

科一考过了科四要再考。

**关键考点：**

- 出血==先止血==：动脉近心端，三角巾固定
- 中毒移出毒区：==抢救==昏迷==先检查呼吸==，==搬运==昏迷先开放气道
- 烧伤==喷冷水==：保鲜膜覆盖，可喝淡盐水

下列行为**都是==错的==**：<span color="red">随意移动、原地等待、包扎细麻绳、软担架</span>

::: example

伤员骨折处出血时，应先固定好肢体==再进行止血==和包扎。

**A.** 正确<br>**B.** 错误

---

出血先止血。

【答案】**B**

:::

::: example

抢救昏迷失去知觉的伤员，应注意什么？

**A.** 马上实施心肺复苏<br>**B.** 使劲掐伤员的人中<br>**C.** 连续拍打伤员面部<br>**D.** 抢救前先检查呼吸

---

【答案】**D**

:::

::: example

移动脊柱骨折的伤员，切勿扶持伤者走动，可使用==软担架==运送。

**A.** 正确<br>**B.** 错误

---

骨折使用软担架会造成二次伤害。

【答案】**B**

:::
